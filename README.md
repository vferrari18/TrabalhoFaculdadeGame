**Firefall 2D** é um jogo de plataforma e tiro (Run and Gun) inspirado no clássico Metal Slug, desenvolvido como parte do trabalho acadêmico para a **UNINTER**.
O jogo apresenta uma temática Sci-Fi com estética Pixel Art, focado em sobrevivência e pontuação.

## 🚀 Funcionalidades
- **Sistema de Combate:** Tiro de plasma com direção baseada no mouse.
- **Inimigos Inteligentes:** Spawns dinâmicos de aranhas cibernéticas (Crystite Spiders).
- **Cenário Dinâmico:** Fundo com efeito Parallax (Céu e Montanhas) e chão texturizado com profundidade.
- **Animações:** Player e inimigos com troca de frames e escalonamento para resolução HD.
- **Gestão de Ranking:** Sistema de salvamento de pontuação local (Kills).
- **Interface Completa:** Menu principal estilizado, tela de input de nome e HUD em tempo real.
- **Trilha Sonora:** Músicas originais em estilo Chiptune/Synthwave e efeitos sonoros de plasma.

## 🎮 Controles
- **W / ESPAÇO:** Pular
- **A:** Mover para a esquerda
- **D:** Mover para a direita
- **MOUSE (Clique Esquerdo):** Atirar Plasma
- **ESC:** Voltar ao menu (na tela de ranking)

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.12+
- **Biblioteca:** Pygame 2.6.1
- **Arquitetura:** Programação Orientada a Objetos (POO)
